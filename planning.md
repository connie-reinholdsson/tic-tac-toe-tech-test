# User Stories

```
As a user,
So I can play the game of Tic Tac Toe,
I want to have an opponent.
```
```
As a user,
So I can play the game of Tic Tac Toe,
Players should be able to take turns until the game is over.
```
```
As a user,
So I can play the game of Tic Tac Toe,
Players should be able to claim a field if it is not already taken.
```
```
As a user,
So I can play the game of Tic Tac Toe,
A turn should end when a player claims a field.
```
```
As a user,
So I can play the game of Tic Tac Toe,
A player should win if they claim all the fields in a row, column or diagonal
```
```
As a user,
So I can play the game of Tic Tac Toe,
The should be over game is over if a player wins or all fields are taken.
```
```
As a user,
So I can play the game of Tic Tac Toe,
The game should announce a winner!
```
